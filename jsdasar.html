Pengenalan Javascript

  1. JavaScript atau sering disingkat JS, pertama kali dibuat tahun 1995 oleh Brandon Eich.
     
  2. Merupakan bahasa pemrograman yang digunakan untuk membangun aplikasi web yang lebih dinamis dan interaktif.
     
  3. JavaScript dikenal dengan bahasa pemrograman FullStack (Backend dan Frontend) karena bisa digunakan untuk membuat aplikasi Backend dan aplikasi Frontend

 

Menjalankan Javascript

# Dari developer tool web browser

  1. Buka browser kemudian pencet f12 pada keyboard untuk membuka developer tools
  2. klik tab console 
  3. Ketikkan script javascriptnya

 

# Embed script pada file HTML

  1. Pada file HTML tambahkan tag <script></script>
  2. Kemudian tuliskan script didalam tag tersebut

 

 

Javascript Variable

Pendeklarasian variable pada javascript bisa menggunakan var,let, & const. Untuk var variable tersebut bisa diakses ke semua block scope sampai ke function, kemudian untuk let variabel tersebut terbatas pada scope jadi variabel tersebut tidak bisa diakses di function scope, berbeda dengan let & var yang nilai variablenya bisa diubah const merupakan variable yang nilainya tidak bisa diubah.

 

var

var nama = “Isyana Sarasvati”;
nama = “Isyana”;
console.log(nama);

 

let

let nama = “Isyana Sarasvati”;
nama = “Isyana”;
console.log(nama);

 

const

const nama = “Isyana Sarasvati”;
console.log(nama);

POPUP

  • Alert digunakan untuk memberi peringatan berupa popup text di browser

    alert('Ini Alert Javascript');

 

  • Prompt digunakan untuk meminta input string dari pengguna browser dalam bentuk popup input text

    prompt('Ketikkan Sesuatu');

 

  • Sedangkan confirm digunakan untuk meminta input boolean dari pengguna browser dalam bentuk popup input pilihan

    confirm('Anda yakin untuk memilih ini?');

 

 

Percabangan

  • IF

    let username="isynsrv";if(username == “isynsrv”)
    {
    console.log("Selamat Datang "+username);
    }

     

  • IF-ELSE

    let username="isynsrv";
    if(username == “isynsrv”)
    {
    console.log("Selamat Datang "+username);
    }
    else
    {
    console.log("Username Kamu Salah")
    }

 

  • ELSE IF

    let username="isynsrv";
    if(username == “isynsrv”)
    {
    console.log("Selamat Datang "+username);
    }
    else if (username == “”)
    {
    console.log("Username Kosong")
    }
    else
    {
    console.log("Username Kamu Salah");
    }
